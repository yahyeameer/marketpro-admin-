## 🛠 Key Features

### 📈 Real-time Analytics
Visualize your performance with interactive charts and live KPI tracking.
- **Revenue Monitoring**: Beautifully rendered area charts for financial trends.
- **Top Performers**: Gamified leaderboard for sales agents.

### 🎯 Lead Pipeline (Kanban)
A futuristic pipeline to manage prospects from inbound to conversion.
- **Drag-ready Layout**: Structured columns for Inbound, Converted, and Lost leads.
- **Detailed Tracking**: Associate leads with field visits and specific agents.

### 💰 Sales Management
Streamlined transaction logging with automated calculations.
- **Record Sales**: Quick-entry form with service selection and date picking.
- **Export Capabilities**: One-click CSV export for reporting.

### 👥 Employee & User Control
Comprehensive management for your workforce and platform access.

## 🚦 Getting Started

### Prerequisites
- Node.js 18.x or later
- A Supabase project

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/marketpro-admin.git
   cd marketpro-admin
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env.local` file in the root:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Launch the engine**
   ```bash
   npm run dev
   ```

4. Deploy!
